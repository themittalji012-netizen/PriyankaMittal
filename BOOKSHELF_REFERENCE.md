# Bookshelf Design Reference - ciangoon.dev

## Key Design Elements Observed

### Background
- **Dark theme** with geometric/3D polygon pattern background
- **Dark navy/charcoal base** with layered geometric shapes creating depth
- Creates a sophisticated, tech-forward aesthetic

### Card Design
1. **Book Cover Display**
   - Large, prominent book cover image (actual book cover art)
   - Takes up ~40-50% of card height
   - High-quality, realistic book imagery

2. **Card Container**
   - Dark semi-transparent background (appears to be dark gray/charcoal with opacity)
   - Subtle border (appears to be a thin light border for definition)
   - Rounded corners (medium-large radius)
   - Clean, minimal padding

3. **Card Information**
   - **Category Badge**: Small label above title (e.g., "FINANCE", "HISTORY")
   - **Star Rating**: 5-star rating display (appears to be gold/yellow stars)
   - **Title**: Bold, white text
   - **Author**: Smaller, secondary text
   - **Description**: Brief 1-2 line description in muted gray text

### Layout
- **Grid**: 4-column layout on desktop
- **Responsive**: Appears to adapt to 2-3 columns on smaller screens
- **Spacing**: Generous gap between cards (appears to be 1.5-2rem)

### Animations & Interactions
- Cards appear to have **hover effects** (likely lift/scale up)
- Smooth transitions on hover
- No excessive animations - keeps focus on content

### Color Scheme
- **Background**: Dark navy/charcoal (#1a1a2e or similar)
- **Card Background**: Slightly lighter dark gray with transparency
- **Text**: White for titles, light gray for descriptions
- **Accents**: Gold/yellow for star ratings
- **Category Badges**: Muted colors matching book categories

### Typography
- **Title**: Bold, sans-serif (appears to be a modern font like Inter or similar)
- **Author**: Regular weight, smaller size
- **Description**: Light weight, small size
- **Category**: Small caps or uppercase, very small

## Implementation Strategy for Priyanka's Portfolio

1. **Adopt dark background** with geometric pattern (optional - can use subtle gradient instead)
2. **Replace gradient book covers** with actual book cover images
3. **Add star ratings** to each book
4. **Improve card styling** with darker semi-transparent backgrounds
5. **Add category badges** with color coding
6. **Enhance hover animations** with lift effect
7. **Increase card spacing** and padding for premium feel
8. **Use actual book imagery** instead of placeholder gradients
